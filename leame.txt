📋 Descripción
Casa SV es el portal líder de bienes raíces en El Salvador, conectando usuarios compradores, agentes vendedores e inmobiliarias en los 262 distritos del país, 14 departamentos, 44 municipios.
✨ Características Principales
🏡 Búsqueda Avanzada - Filtros por tipo, precio, ubicación, características
🗺️ Mapa Interactivo - Visualización de propiedades con Leaflet
💰 Subastas - Pujas desde $4 dólar
🏦 Calculadora Hipotecaria - Estimación de cuotas con bancos de El Salvador
🔗 Enlaces Útiles - Bancos, cooperativas, alcaldías, servicios públicos
🌙 Modo Oscuro - Tema claro/oscuro
📱 Responsive - Optimizado para móvil, tablet y desktop
---
🛠️ Tecnologías
·	Frontend: React + TypeScript + Vite
·	Estilos: Tailwind CSS + shadcn/ui
·	Estado: Zustand
·	Mapas: Leaflet + React-Leaflet + google earth 
·	Iconos: Lucide React

📁 Estructura del Proyecto
```
app/
├── src/
│   ├── components/     # Componentes reutilizables
│   ├── pages/         # Páginas principales
│   ├── data/          # Datos (propiedades, distritos, enlaces)
│   ├── store/         # Estado global (auth)
│   ├── hooks/         # Custom hooks
│   └── types/         # Tipos TypeScript
├── public/            # Assets estáticos
└── dist/             # Build de producción
```
